API de Gerenciamento de Frota de Caminhões

Bem-vindo à API de Gerenciamento de Frota de Caminhões

Esta API fornece endpoints para gerenciar uma frota de caminhões, incluindo cadastro, listagem e atualização de informações, com integração à API FIPE para validação de dados e obtenção de valores de referência.

Console de administração do banco de dados H2 (em memória). Use "sa" como usuário e deixe a senha em branco para acessar.
API de Caminhões
Endpoint principal para listar todos os caminhões cadastrados no sistema.

Endpoints Principais

  • GET /api/trucks - Lista todos os caminhões
  • GET /api/trucks/{id} - Busca um caminhão pelo ID
  • POST /api/trucks - Cadastra um novo caminhão
  • PUT /api/trucks/{id} - Atualiza um caminhão existente
