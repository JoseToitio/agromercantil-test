Sistema de Gerenciamento de Frota de Caminhões

Bem-vindo ao Sistema de Gerenciamento de Frota de Caminhões

Este sistema é composto por duas partes:

1. Backend (API REST)

O backend fornece os endpoints para gerenciar os caminhões e integração com a API FIPE.

Acessar Backend

2. Frontend (Interface de Usuário)

O frontend é a interface de usuário para interagir com o sistema.

Acessar Frontend

Instruções de Uso

  1. Certifique-se de que o backend está em execução na porta 8080
  2. Certifique-se de que o frontend está em execução na porta 4200
  3. Acesse o frontend para começar a usar o sistema

Problemas Comuns

  • Erro 404 ao acessar o backend: Verifique se o servidor Spring Boot está em execução
  • Problemas ao acessar o frontend: Verifique se o servidor HTTP está em execução
